# fitbit_apps
Developing apps with Virtual Studio Code:
<b> Build new project </b>
* npx create-fitbit-app <project-name>
Check dependencies:
1. `"devDependencies": {
  "@fitbit/sdk": "~5.0.0",
  "@fitbit/sdk-cli": "^1.7.3"
}`
2. `npm install`
3. `npx fitbit-build`
4. `npx fitbit`

Useful commands for fitbit-CLI:
* `fitbit$ build`
* `fitbit$ install`
* `fitbit$ bi` (build & install)
